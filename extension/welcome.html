Chrome extension setup

Capture the current tab and queue it in YDL Studio.

Keep the local app running, choose a format from the toolbar popup, and send videos or audio into your download queue without copying links manually.

Open extensions
01

Load the folder

Open chrome://extensions, turn on Developer mode, and choose the extracted extension folder.

02

Pin the button

Pin YDL Studio Capture to the Chrome toolbar so it is always one click away.

03

Queue downloads

Open a supported media page, select MP4, MP3, M4A, or Best, then start the download.

Production checks

Before using the extension

  • YDL Studio is open at http://127.0.0.1:5179.
  • The extension has permission to read the current tab URL.
  • The media page is public or already signed in inside Chrome.
